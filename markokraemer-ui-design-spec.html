01 — Overview

Site Concept & Architecture

A sci-fi terminal emulator as a personal portfolio site. The entire experience is framed as a "Kortix Neural Interface" — a fictional OS that presents Marko Kraemer as an AI model.

C Core Concept

The site is a single-page React application that simulates a macOS-style terminal window floating on a deep-space starfield background. The "Kortix Neural Interface" theme turns a standard portfolio into a memorable sci-fi experience.

S Starfield Background Specification

06 — Animations & Transitions

Motion Design

The animations sell the "booting up" narrative. Everything should feel like a system initializing.

A Animation Sequence Timeline

0ms — Page Load
Starfield background renders immediately (dark space with stars)
~200ms — Terminal Fade-In
Terminal window fades in with CSS class .fade-in. Use opacity: 0 → 1 over ~400ms ease-out.
~600ms — Welcome Text Typewriter
Text begins "typing" character by character. Speed: ~20-40ms per character. Each line appears sequentially.
~2s — Status Dashboard
The Cognitive Status block types out row by row. Two-column data renders with precise monospace alignment.
~4s — Menu Appears
"Select an option:" and "1. Activate Voice Interface" type out. Cursor begins blinking, input enabled.
User Action — Option Selected
User types "1" + Enter → Response types out: "You chose to activate..." + "Connecting..." with animated dots.

T Typewriter Effect

JS Logic // Character-by-character reveal const typeText = async (text, delay = 30) => { for (const char of text) { appendChar(char); await sleep(delay); } }; // Variation: line-by-line for faster sections const typeLine = async (line, delay = 80) => { appendLine(line); await sleep(delay); };

C Cursor Blink

CSS @keyframes blink { 0%, 100% { opacity: 1; } 50% { opacity: 0; } } .cursor { animation: blink 1s step-end infinite; color: #FFFFFF; } /* Block cursor character: █ (U+2588) */

S Star Twinkling

CSS / Canvas /* Each star pulses opacity independently */ @keyframes twinkle { 0%, 100% { opacity: 0.3; } 50% { opacity: 1; } } .star { animation: twinkle 3s ease-in-out infinite; animation-delay: var(--delay); /* randomized per star */ } /* Or use Canvas for better performance with many stars */

A Accessibility Notes

While the terminal aesthetic is inherently visual, ensure:

• Hidden input has proper aria-label ("Terminal command input")

• Screen readers can access the text content

• prefers-reduced-motion disables typewriter (show text instantly)

• Sufficient contrast: green on black passes WCAG AA

P Performance Notes

Key optimizations:

• Starfield: use requestAnimationFrame, cap at 30fps

• Typewriter: batch DOM updates, avoid per-character reflows

• Font: preload Anonymous Pro via <link rel="preload">

• Total JS bundle target: < 80KB gzipped
